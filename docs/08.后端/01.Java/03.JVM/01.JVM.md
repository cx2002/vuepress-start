# 													JVM



## 1. JVM的内存模型(JMM)

JMM是一种抽象的概念,并不真实存在,描述的是一组规范或规则来**屏蔽**`各种硬件和操作系统间的内存访问差异`,以实现上Java程序在任何平台上都能达到一致的内存访问效果

JMM的主要**目的**就是`定义程序中各种变量的访问规则`(即关注将程变量值存储至内存和从内存中取出变量值这样的底层细节)
